# Smart Water Tank Management using Reinforcement Learning

A reinforcement learning–based intelligent water tank management system designed to optimize pump operations for a dual water tank setup under varying water demand and electricity pricing conditions using Q-learning.

---

## Features
- Q-learning–based adaptive pump control
- Intelligent dual water tank level management
- Dynamic demand and electricity cost handling
- Custom reward function optimization
- Policy heatmap visualization
- Reward convergence analysis

---

## Tech Stack
- Python
- Reinforcement Learning
- Q-Learning
- NumPy
- Matplotlib
- Seaborn

---

## Project Workflow
1. Environment simulation for dual water tank system
2. State and action space design
3. Reward function implementation
4. Q-table training and updates
5. Policy evaluation and visualization

---

## 📊 Model Highlights
- Optimized water tank levels under dynamic conditions
- Reduced unnecessary pump operations
- Achieved stable policy convergence through simulation
